BOOTSTRAP KLARA HAGELIN

HTML Element

HTML Element är allting som har en start och en slut tag. TEX h1 och p

HTML (generellt)

Ett märkspråk för hypertext som är standard för hemsidor. HTML står för hypertext markup language. Med HTML kan man infoga information och ange struktur men om man vill styla måste man komplettera med TEX CSS.

ID

Man kan ge ett html element ett specifikt ID för att kunna styla det specifika ID utan att ändra allt med samma element. Endast ett html element kan ha samma ID.

Class

Class är liknande ett ID förutom att flera HTML element kan dela en klass. Class används för att kunna styla alla element med samma klass utan att ändra allt med de elementen.

Meta (ordet)

Meta är en beskrivelse av något. Till exempel metadata som är information/data om datan.

Responsive

Responsive betyder att sidan anpassar sig efter alla enheter och automatiskt ändras beroende på skärmstorlek. Används för att en hemsida ska se bra ut på alla sorters enheter.

Http

HTTP står för hypertext transfer protocol och är ett protokoll för hur webbsidor ska överföras över internetet WWW.

Https

Som http fast krypterat. Står för hypertext transfer protocol secure.

DNS

DNS står för Domain Name System. DNS används för att dirigera internet trafik. DNS kopplar ihop en IP-adress med ett domän namn.

CDN

CDN står för Content Delivery Network och är proxyservrar som är utspridda över världen för att vara så nära användarna som möjligt. Detta gör att webbsidor och liknande kan levereras till användare mycket snabbare.

Compressed data

Compressed data eller komprimerad data är data som har blivit för minskad. Datan är omkodad för att kräva mindre bits för att presenteras, detta gör att datan tar mindre plats på disken.

JQuery

JQuery är ett JavaScript-bibliotek för att förenkla bland annat HTML och CSS kodning.

Flexbox

Flexbox är en webblayout modell i css som är responsiv utefter skärmen. Detta gör att elementen inuti containern ordnas automatiskt efter skärmstorleken.

Grid

Ett annat sätt att att göra en responsiv layout i css. Grid skapar ett rutnät.

Code snippets

Code snippets är små utdrag ur en kod som ofta används som mall till delar av koden som ofta repeteras.

WASM

WASM eller Webassembly är ett programmeringsspråk som används som en förbättring till javascript men det är inte en ersättning utan ett samarbete mellan de två språken.

Chromium

Chromium är en bas för webbapplikationer som ofta används för att bygga upp nya applikationer. Chromium kan även användas som en webbläsare och funkar ungefär som chrome eftersom chrome är baserat på chromium.

SVG

SVG - scalable vector graphics är ett vektorbaserat format för 2D bilder men även för animationer och interaktiva bilder.

png24

Png24 är ett bildformat för kompression som stödjer 24-bitars färger.
